# Coin Detection using OpenCV in Python

## Overview

This project demonstrates coin detection using OpenCV and image processing techniques. The program reads an image containing coins, preprocesses it, and detects the coins using computer vision algorithms.

## Features

* Load and process coin images
* Convert images to grayscale
* Apply image filtering and thresholding
* Detect coin boundaries using OpenCV
* Draw circles around detected coins
* Display the total number of coins detected

## Technologies Used

* Python
* OpenCV
* NumPy
* Matplotlib
* Jupyter Notebook

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Aadi6777/Coin-Detection-using-OpenCV-in-Python.git
```

2. Navigate to the project directory:

```bash
cd Coin-Detection-using-OpenCV-in-Python
```

3. Install required packages:

```bash
pip install opencv-python numpy matplotlib
```

## Usage

1. Open the Jupyter Notebook:

```bash
jupyter notebook
```

2. Open:

```text
Coin-Detection-workfile.ipynb
```

3. Run all cells to detect coins in the provided image.
